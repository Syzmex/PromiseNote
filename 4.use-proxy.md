# 在项目中使用 Proxy 

修改配置文件 `.roadhogrc`，**改名为** `.roadhogrc.js`，内容改成 **Javascript** 格式

```diff
+ var proxy = requeir( './proxy.config.js' );

- {
+ module.exports = {
    ...
+   "proxy": proxy
    ...
- }
+ };
```

新增文件 `proxy.config.js`

```js
// 样例
// /api/users => http://jsonplaceholder.typicode.com/user
module.exports = {
  "/api": {
    "target": "http://jsonplaceholder.typicode.com/",
    "changeOrigin": true,
    "pathRewrite": { "^/api" : "" }
  }
};
```
